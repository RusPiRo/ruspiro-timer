# Timer RusPiRo crate

## Usage
To use the crate just add the following dependency to your ``Cargo.toml`` file:
```
[dependencies]
ruspiro-timer = "0.0.1"
```

Once done the access to the timer functions is available in your rust files like so:
```
use ruspiro_timer;


```
